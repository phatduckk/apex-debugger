How it works

Statements evaluate top-to-bottom. The last true statement wins and determines the physical output state.
Status refreshes every 5 s from the Apex. Colors update instantly as you type.

Evaluable — If / Then

If Time HH:MM to HH:MM Then ON|OFFTrue if current time is in range. Midnight-spanning ok (e.g. 21:00 to 03:00). Inclusive both ends.
If DOW SMTWTFS Then ON|OFF7 chars S M T W T F S. Hyphen = skip day. e.g. -MTWTF- = weekdays only.
If [Probe] > [val] Then ON|OFFTemp, pH, ORP, Salt, or any named probe — greater-than comparison.
If [Probe] < [val] Then ON|OFFSame probe, less-than comparison.
If [Input] OPEN Then ON|OFFDigital switch — OPEN = value 0.
If [Input] CLOSED Then ON|OFFDigital switch — CLOSED = value 200.
If Output [name] = ON|OFF Then ON|OFFTests another outlet's current on/off state.
If Outlet [name] = ON|OFF Then ON|OFFIdentical to Output (older firmware keyword).
If Output [name] Percent > [val] Then ON|OFFTests variable output intensity 0–100.
If Output [name] Percent < [val] Then ON|OFFSame, less-than.
If FeedA|B|C|D MMM Then ON|OFFTrue if that feed cycle is active. MMM = delay minutes after cycle ends.
If Error [name] Then ON|OFFTrue if the named outlet is in ERR state (overload / short circuit).
If Sun [+/-MMM]/[+/-MMM] Then ON|OFFTrue if current time is between (sunrise + offset1) and (sunset + offset2). Data from /rest/config.
If Moon [+/-MMM]/[+/-MMM] Then ON|OFFTrue if current time is between (moonrise + offset1) and (moonset + offset2). Data from /rest/config.
If [Probe] < RT+[val] Then ON|OFFCompares probe to seasonal regional temp ± differential. e.g. RT+0.4, RT+-0.4, RT+. Data from /rest/config.

Always executes

Set ON|OFFUnconditionally sets the outlet register. Always green — participates in winner selection.
Set [Profile]Sets a named profile. Treated as neutral — profile state can't be evaluated.
Fallback ON|OFFWhat the physical outlet does if communication with the base unit is lost. Neutral.

Cannot evaluate — always grey

OSC MMM:SS/MMM:SS/MMM:SSOscillate. Three segments: delay / on-time / off-time. Requires cycle-phase history.
Defer MMM:SS Then ON|OFFDelays a state change until the register holds that state for the full duration. Requires outlet history.
Min Time MMM:SS Then ON|OFFForces outlet to hold its current state for minimum duration before switching. Requires outlet history.
When ON|OFF > MMM:SS Then ON|OFFForces outlet to manual OFF if it has been in the specified state longer than the duration. Requires runtime tracking.
If Power Apex|EB ON|OFF MMMTests power state of base unit or Energy Bar. Partial data only — grey for now.
